📘 UC12 – Apply Tags to Contacts

Overview

UC12 extends the tagging system by allowing logged-in users to assign one or multiple tags to contacts.
This use case demonstrates the association between Contact and 
Tag and uses the Observer Pattern to notify the UI whenever tags are applied or removed.

Features

Apply one or more tags to a contact.
Remove tags from a contact.
Maintain a many-to-many relationship between contacts and tags.
Automatic UI updates when tags change via the Observer Pattern.
Use of Java Set for uniqueness and bidirectional relationship management.
